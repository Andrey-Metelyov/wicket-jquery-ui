The provided list should be modifiable for the new order to be reflected on 'sort' event. That means, the use of Arrays.asList or Collections.unmodifiableList should be avoided.
If you do not want that the new order is reflected to the model object, simply remove the call to <code>super.onSort(AjaxRequestTarget, String, int)</code>
