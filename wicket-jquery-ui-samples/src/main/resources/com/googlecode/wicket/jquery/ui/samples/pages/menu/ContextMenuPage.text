<code>ContextMenu</code> is triggered when a <code>Component</code> holding a <code>ContextMenuBehavior</code> fires the 'contextmenu' event.

<h4>Known limitation</h4>
The <code>Component</code> cannot be a 'clickable' element (such as a <code>link</code> for instance).
